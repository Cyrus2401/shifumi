# 🎯 Shifumi

Un jeu classique de Pierre-Papier-Ciseaux contre l'IA, simple et rapide !

🔗 **[Jouer maintenant](https://shifumi-sigma.vercel.app/)**

## 🚀 Comment jouer ?
1. Choisissez le nombre de manches gagnantes (1 à 10).
2. Cliquez sur 🪨, 📄 ou ✂️.
3. Le premier à atteindre le score cible remporte la partie !

## 🛠️ Technologies
- HTML, CSS, JavaScript (Vanilla)
